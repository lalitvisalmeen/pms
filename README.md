# Project Management System
A full-stack Project Management System built using Laravel and React that allows users to manage projects, tasks, and workflows efficiently. This project was developed as part of hands-on learning to deepen my understanding of React integration with a Laravel backend while designing a scalable application architecture.

The application enables users to create projects, manage tasks, assign responsibilities, and track progress throughout the task lifecycle.

# Technologies used
Backend
PHP 8
Laravel 12
MySQL

Frontend
React
TypeScript
TailwindCSS
# Features
1. Create and manage projects
2. Create, update, and delete tasks
3. Assign tasks to users
4. Track task status and progress
5. Form validation and error handling
6. Dynamic UI using reusable React components

# Purpose of the Project
The goal of this project was to gain practical experience building a full-stack application using Laravel and React. It focuses on understanding component-based frontend architecture, and relational database design.
